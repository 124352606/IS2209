About

CryptoTrack — built for IS2209.

What it does

CryptoTrack is a live cryptocurrency price tracker. It pulls real-time prices from the CoinGecko API for Bitcoin, Ethereum, Solana, Cardano, and Dogecoin, and displays them on a live dashboard. Users can add coins to a personal watchlist, which is persisted in a PostgreSQL database. A built-in currency converter lets you calculate the USD value of any coin amount instantly.
